# RVT Freight Ledger

A comprehensive freight management and ledger application designed to streamline logistics operations, financial tracking, and fleet management.

## Overview

RVT Freight Ledger provides a robust interface for trucking companies and freight brokers to manage their loads, expenses, and profitability in real-time. It replaces manual spreadsheets with a modern, data-driven dashboard.

## Key Features

- **Freight Management**: detailed tracking of every load, including pickup/drop logistics, cargo specifications (NMFC, Freight Class, Temp Control), and hazardous material flags.
- **Financial Dashboard**: Automatic calculation of revenue per load, including line haul, fuel surcharges, and accessorials.
- **Profitability Analysis**: Real-time net profit calculation after accounting for driver pay, owner percentages, and operational expenses.
- **Asset & Driver Tracking**: Management of fleet assets (trucks, business cars) and driver profiles with customizable pay structures.
- **Recycle Bin**: Integrated soft-delete functionality to safely restore accidentally removed records.
- **Responsive Design**: Built for desktop and tablet use, ensuring access from the office or the road.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context / Hooks
- **Forms**: React Hook Form with Zod validation
