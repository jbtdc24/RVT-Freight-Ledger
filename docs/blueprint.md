# **App Name**: RVT Accounting

## Core Features:

- Freight Ledger: Form and table to track freight IDs, logistics, and financials, including revenue per trip and auto-calculated RPM.
- Business Expenses: Section to log maintenance, fuel, and repairs with description/comment field.
- Asset Management: List and track trucks and business cars.
- Payroll Calculator: A calculator to track driver pay based on completed loads.
- Net Profit Dashboard: Summarize total business income vs. expenses for a net profit view.

## Style Guidelines:

- Dark background: Dark purple (#301934) to set a modern and professional tone, reducing eye strain during prolonged use.
- Primary color: Electric purple (#BF40BF) to create vibrancy, reflecting the dynamic nature of the trucking business.
- Accent color: Light blue (#87CEFA) for interactive elements and highlights, to complement the purple tones and draw user attention.
- Body font: 'Inter' sans-serif with a modern, machined, objective, neutral look; suitable for body text.
- Headline font: 'Space Grotesk' a proportional sans-serif with a computerized, techy, scientific feel; suitable for headlines
- Use minimalist icons for navigation and key features, maintaining a clean and intuitive user experience.
- Incorporate 'Manual Entry' buttons for all sections to accommodate various data input preferences. Data should persist in a database.
- Use subtle transitions and animations to enhance user engagement without overwhelming the interface.
